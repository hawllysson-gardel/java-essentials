# Java Essentials

Bem-vindo ao repositório Java Essentials! Este projeto abrange conceitos essenciais de Java, com exemplos e explicações para ajudá-lo a entender e aplicar esses conceitos em seus próprios projetos.

## Índice

1. **Access Modifiers**
2. **Annotations**
3. **Data Structures**
4. **Design Patterns**
5. **DRY, KISS and YAGNI**
6. **Encapsulation**
7. **Exceptions**
8. **Keyword Final**
9. **Garbage Collector**
10. **Generics**
11. **Inheritance**
12. **Interfaces**
13. **Lambdas**
14. **Polymorphism**
15. **SOLID**
16. **Streams**
